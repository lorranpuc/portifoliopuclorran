## 📖 About the Project

This is my personal portfolio website, designed to showcase my professional journey, technical skills, and main projects. As a **Mid-level Python Developer & RPA Specialist**, this space serves as a central hub for my work, ranging from complex enterprise legal automations (RPA) to intelligent bots and cloud infrastructure (AWS/GCP).

The layout is fully responsive, featuring a modern sidebar navigation and a clean UI, making it easy for recruiters and developers to explore my background and reach out.

## ✨ Features

- **Responsive Design:** Adapts perfectly to desktops, tablets, and mobile devices.
- **Dynamic Navigation:** Retractable sidebar menu for a clean user experience.
- **Projects Showcase:** Detailed sections highlighting open-source bots and enterprise-grade RPA solutions.
- **Working Contact Form:** Integrated with [StaticForms](https://www.staticforms.xyz/) to receive emails directly from the site without a backend.
- **Modern Iconography:** Integration with Bootstrap Icons and Devicon for high-quality tech stack logos.

## 🚀 Technologies Used

This project was built using the following technologies:

- **HTML5** & **CSS3** (Custom styling and animations)
- **JavaScript** (Vanilla JS for DOM manipulation and menu toggling)
- **[Bootstrap 5.3](https://getbootstrap.com/)** (Grid system and utilities)
- **[Bootstrap Icons](https://icons.getbootstrap.com/)** (General UI icons)
- **[Devicon](https://devicon.dev/)** (Programming language and tools icons)
